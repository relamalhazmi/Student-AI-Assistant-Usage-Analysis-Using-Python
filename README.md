# Student-AI-Assistant-Usage-Analysis-Using-Python
Data analysis of 10,000 student AI interactions using Python to identify usage patterns, cleaning 45 duplicates, and analyzing satisfaction levels

## Project Overview
This project analyzes a dataset of **10,000 student sessions** to understand how students (High School to Graduate) interact with AI assistants like ChatGPT.

## Key Objectives
* **Analyze** student satisfaction and retention (70.7% return rate).
* **Identify** common tasks (Writing is #1, Research is least used).
* **Visualize** usage patterns across different academic levels.

## Top Insights
* **Undergraduates** are the most frequent users.
* Average satisfaction rating is **3.41/5**.
* **70.7%** of students are repeat users, showing high trust in AI tools.

## Tools Used
* **Python** (Pandas, Matplotlib, Seaborn).
* **Dataset Source:** Kaggle (Synthetic Student Life Dataset).
